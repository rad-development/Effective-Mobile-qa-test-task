## 4. Ожидаемые коды ответа (status codes)

- **Успешное создание пользователя**  
  `201 Created` – пользователь успешно создан. В теле ответа возвращаются все поля, включая `id` и `status`.

- **Ошибка валидации данных**  
  `400 Bad Request` – отправлены некорректные данные (например, пустое имя, неверный email, возраст < 0). Сервер возвращает сообщение об ошибке.

- **Попытка создать пользователя с уже существующим email**  
  `409 Conflict` – email уже зарегистрирован. Сервер возвращает сообщение о конфликте ресурса.
